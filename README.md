# 文科期末复习 Skill

一个用于 Claude 的 Skill，帮助你在 3-7 天内快速啃下一门文科课程的期末考试。

适用范围：文学、历史、哲学、传播学、比较文学等任何以"文字资料量大、需要理解+背诵+答题"为特点的文科课程。

---

## 这个 Skill 解决什么问题

期末复习时，大部分时间被浪费在这些事情上：

- 从头读一遍课本和PPT，分不清哪些是重点
- 自己画思维导图、整理框架，耗时但低效
- 不知道该背什么、怎么背
- 想刷题但不知道可能考什么

这个 Skill 会代替你完成"整理"这一步，把你的时间留给"理解"和"记忆"。

---

## 这个 Skill 能做什么（完整流程）

把你的课程资料和考试要求发给 Claude 后，它会按以下六步帮你整理：

### 1. 课程认知建立
先告诉你这门课整体在讲什么、核心问题是什么、各模块之间是什么关系，并配一张思维导图，让你对整门课先有一个全局认识，而不是一上来就陷入细节。

### 2. 知识点拆解
逐模块梳理具体内容：每个概念用一句话说清楚核心定义，容易混淆的概念做对比表格，晦涩的表述会用更易懂的语言重新解释，必要时补充例子帮助理解（补充内容会明确标注，不会和课本内容混淆）。

### 3. 重要度判断 + 题型预判
结合你提供的考试要求（题型、分值、命题范围、命题原则等），告诉你：
- 哪些内容是必考、重要、还是了解即可
- 每个知识点应该精确背诵、理解记忆、还是大致了解
- 每个知识点最可能出现在哪种题型里
- 哪些内容根据老师说明不会考，可以直接跳过

### 4. 备考资料整理
直接产出可以拿来背诵和练习的材料：
- 判断题/选择题的易错点和正确认知
- 名词解释的参考答案（严格按照你的考试要求，比如"以课本为准、不要发散"）
- 简答题的分点答案要点
- 论述题的可能方向和答题框架（如果老师说论述题不用专门准备，会改为只整理素材和思路，不会逼你背答案）

### 5. 学习规划
根据你距离考试还有几天，制定具体到每天的学习安排，并融入科学的复习节奏（学完后短期回顾、隔天复习等），3天和7天的安排都可以。

### 6. 反馈与答疑
你做练习、答题后，Claude会告诉你哪些答对了、漏了什么、可以怎么补充。复习过程中随时可以提问，Claude会直接回答，并告诉你这个知识点的重要程度和考法。

## 如何使用

### 方法一：在 Claude.ai 中上传 Skill（推荐）

大致步骤如下（界面文字可能随版本略有变化）：

1. 下载本仓库中的 `humanities-exam-prep.skill` 文件
2. 打开 Claude.ai，进入 **设置（Settings） → Capabilities**，确认"Code execution and file creation"（代码执行与文件创建）已经打开。Skills 功能依赖这个开关，没打开的话上传入口可能找不到或不生效
3. 进入 **Customize → Skills**，找到上传/添加自定义 Skill 的入口，选择 `humanities-exam-prep.skill` 文件上传

   > 注意：如果上传时提示文件格式不支持，尝试把文件后缀从 `.skill` 改成 `.zip` 后再上传——这个文件本质上就是一个 zip 压缩包，只是后缀不同。
4. 上传成功后，开启新对话，把你的课程资料（PPT文字、教材摘录、笔记等）和老师给的考试要求一起发给 Claude，并说明"帮我复习这门课"
5. Claude 会自动按上面的六步流程开始整理

如果按以上步骤找不到入口，建议直接在 Claude.ai 内搜索"如何使用 Skills"获取最新的官方指引，因为这个功能比较新，界面位置可能会调整。

### 方法二：没有 Skill 上传功能时（手动方式）

如果你的账号类型不支持上传 Skill（例如某些受限场景），也可以手动使用：

1. 解压 `humanities-exam-prep.skill` 文件（它本质上是一个 zip 压缩包，可以用解压软件打开）
2. 解压后会看到一个 `SKILL.md` 文件，打开它，把里面的全部文字内容复制
3. 把复制的内容粘贴到和 Claude 的新对话开头，作为给 Claude 的指示
4. 然后正常发送你的课程资料和考试要求

手动方式的效果和正式安装 Skill 基本一致，只是每次开新对话都需要重新粘贴一次指示文本。

## 使用建议

**资料尽量完整**：PPT文字内容、教材重点段落、课堂笔记，能提供的都提供。资料越完整，整理出来的内容越准确。

**一定要提供考试要求**：如果老师说明了题型、分值分布、考察范围、是否有内容不考等信息，一定要告诉 Claude。这是"重要度判断"这一步最关键的依据，没有这些信息，Claude 只能基于课程内容做大致预判，并会明确告知你这是预判而非确定结论。

**告诉 Claude 还剩几天**：方便制定合理的学习规划，3天和7天的安排会完全不同。

**资料量大时分批发送**：如果一门课的资料特别多，可以先发一部分，让 Claude 先建立整体框架，再分模块继续发送其余资料。

**遇到不准确的地方直接反馈**：比如某个知识点的重要度判断你觉得不对，或者某个名词解释写得太啰嗦，直接告诉 Claude，它会调整。这是一个可以持续优化的过程。

## 文件说明

`humanities-exam-prep.skill` 是这个 Skill 的主文件，里面包含：

- 核心的六步工作流程
- 针对文学类课程的专项整理方法（人物/流派/作品分析格式、影响研究答题框架等）
- 针对历史类课程的专项整理方法（时间线整理、原因分析答题框架等）
- 针对哲学/理论类课程的专项整理方法（概念辨析、理论体系梳理等）
- 一套可以直接复制使用的 AI 提示词库（适合手动使用方式）
