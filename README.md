# 文科期末复习 Skill

一个用于 Claude 的 Skill，帮助你在 3-7 天内快速啃下一门文科课程的期末考试。

适用范围：文学、历史、哲学、传播学、比较文学等任何以"文字资料量大、需要理解+背诵+答题"为特点的文科课程。

## 这个 Skill 能做什么

把你的课程资料（PPT内容、教材摘录、课堂笔记等）和考试要求发给 Claude，它会按以下流程帮你整理：

1. **课程认知建立**——先告诉你这门课讲了什么、整体框架是什么样（配思维导图）
2. **知识点拆解**——逐模块梳理重点内容，必要时补充解读和例子
3. **重要度判断 + 题型预判**——结合你的考试要求，告诉你什么必考、什么不用看
4. **备考资料整理**——直接产出名词解释、简答要点、判断题易错点、论述题方向等
5. **学习规划**——根据剩余天数制定具体的每日学习安排
6. **反馈与答疑**——你练习后给反馈，随时回答你的问题

## 如何使用

### 方法一：在 Claude.ai 中使用（推荐）

1. 下载本仓库中的 `humanities-exam-prep.skill` 文件
2. 在 Claude.ai 的设置中找到 Skills（技能）功能，上传这个文件
3. 之后在对话中，把你的课程资料和考试要求发给 Claude，告诉它"帮我复习这门课"，它会自动按这个流程帮你整理

### 方法二：没有 Skill 上传功能时

如果你使用的 Claude 版本不支持上传 `.skill` 文件，可以：

1. 解压 `humanities-exam-prep.skill` 文件（它本质是一个 zip 压缩包）
2. 打开里面的 `SKILL.md`，把全部内容复制粘贴到对话开头
3. 然后正常发送你的课程资料和考试要求

## 使用建议

- **资料越完整越好**：PPT内容、教材摘录、课堂笔记都可以提供
- **务必提供考试要求**：题型、分值、范围、命题原则等信息，能大幅提升整理的针对性
- **告诉 Claude 剩余天数**：方便制定合理的学习规划
- **遇到不准确的地方直接反馈**：这是一个迭代过程，Claude 会根据你的反馈调整
